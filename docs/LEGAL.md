# Legal

Madrador Film est un projet de catalogue, de diagnostic et d'integration de providers Nuvio.

Regles du projet :

- aucune video n'est stockee dans ce depot ;
- aucun contournement DRM n'est autorise ;
- aucun contournement de paywall n'est autorise ;
- aucun cookie prive, session personnelle ou acces cache ne doit etre ajoute ;
- les providers instables ou non responsables doivent rester desactives ;
- les demandes de retrait doivent etre traitees rapidement.

Si un provider exige une authentification, un cookie prive ou une protection, il ne doit pas etre active.
